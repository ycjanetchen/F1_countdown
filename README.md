## ✨ Features

- **Lights-out splash screen** — animated F1 start lights sequence before every session
- **Driver selector** — pick any of the 22 drivers across all 11 teams for the 2026 season
- **Live countdown** — real-time days / hours / minutes / seconds to the next race, with per-second spark animations
- **Full 2026 race calendar** — all 22 rounds including sprint weekends, with animated race list
- **Timezone support** — 14 preset timezones plus automatic local timezone detection
- **Team-colour theming** — the entire UI adapts to your selected driver's team colour
- **Aurora background** — WebGL-powered animated aurora effect on every screen
- **Zero dependencies** — pure HTML, CSS, and vanilla JavaScript in a single file

## 🚀 Getting Started
Click this link https://ycjanetchen.github.io/F1_countdown/

No installation or build step required.

```bash
# Clone the repo
git clone https://github.com/YOUR_USERNAME/f1-countdown-2026.git

# Open directly in your browser
open f1_2026_gradient_btn.html
```

Or just download the HTML file and double-click it. That's it.
